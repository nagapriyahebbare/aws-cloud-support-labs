# S3 AccessDenied Troubleshooting Lab

## Objective
Simulated and resolved AWS S3 AccessDenied errors using IAM policies and AWS CLI.

## Services Used
- Amazon S3
- AWS IAM
- AWS CLI

## Problem
IAM user could not upload files due to missing S3 permissions.

## Resolution
Attached AmazonS3FullAccess policy and verified successful upload.

## Skills Learned
- IAM troubleshooting
- S3 permissions
- AWS CLI
- Cloud debugging
