## 🎯 Why This Repo?

AWS Cloud Support Engineers spend most of their time diagnosing exactly these types of issues.

This repo is built to:
- Practice structured root cause analysis (RCA)
- Build muscle memory for AWS CLI diagnostic commands
- Document troubleshooting in the same format used in real support tickets
